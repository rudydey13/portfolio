def distro(data):
    nb_sample, nb_var = data.shape
    labels = list(data)
    plt.figure(figsize = (12,8))
    for i in range(nb_var):
        ax = plt.subplot(3,3, i + 1)     
        data_col = data.iloc[:, i]
        data_col.hist()
        plt.title(f'{labels[i]}')
        ax.title.set_fontsize(30)
        for item in ([ax.xaxis.label, ax.yaxis.label] +
              ax.get_xticklabels() + ax.get_yticklabels()):
            item.set_fontsize(20)
    plt.tight_layout(pad=2,w_pad=2,h_pad=2)
    plt.show()

Generate a Linear Prediction Model Using Elastic Net Regression¶

"Elastic Net" combines characteristics of both Ridge and Lasso; ie, it reduces the impact of different features while not eliminating all of the features. The Elastic Net is defined as:

$$ \begin{align*} \min_{\beta}& \Bigg\{\underbrace{\sum_{i=1}^n\big(y_i-\hat{y}_i\big)^2}_{\text{RSS}} + \lambda\times\bigg[\frac{(1-\alpha)}{2} \times\left(\sum_{j=1}^p \beta_j^2\right) + \alpha \times\sum_{j=1}^p | \beta_j|\Bigg\}\bigg] \end{align*} $$

The parameter $\alpha$ is the mixing parameter between Ridge ($\alpha=0$) and Lasso ($\alpha=1$), while $\lambda$ plays the same role as the penalty parameter. Note: the routines in sklearn (annoyingly) refer to $\alpha$ as the penalization paramater but convention is to use $\lambda$ as the penalization parameter.

  • Fit the model using only the training data.
  • Search for the best mixing parameter ($\alpha$ above) using "l1_ratio=l1_grid" (documentation) where 
    l1_grid = np.arange(.1,1,1e-2).tolist()
  • Conduct cross-validation via the 'leave-one-out' approach. See documentation.
  • Save the coefficients.
  • Generate and save the mean squared error (mse) using the training data.
  • Generate and save mean squared error (mse) using the testing data.
